# Sinfra TaxiGo Web Application

## Description
Sinfra TaxiGo is a web application designed for taxi services with features like GPS tracking, client and driver interfaces, an admin dashboard, a pricing system, voice commands, and offline support.

## Features
- **GPS Tracking**: Real-time tracking of taxis.
- **Client Interface**: User-friendly interface for clients to book rides.
- **Driver Interface**: Interface for drivers to manage rides and earnings.
- **Admin Dashboard**: Comprehensive dashboard for admins to oversee operations.
- **Pricing System**: Dynamic pricing based on demand and distance.
- **Voice Commands**: Integration of voice commands for ease of use.
- **Offline Support**: Functionality in low connectivity scenarios.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sinfrataxigo/sinfra-taxigo.git
   ```
2. Navigate to the directory:
   ```bash
   cd sinfra-taxigo
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```
